AMP Panel is a free game server hosting service provided to the VR creator community.

## Service Overview

AMP Panel offers hosting for:
- **Resonite Headless Servers** - Host your own Resonite sessions
- **Game Servers** - Various game server hosting options
- **File Sharing** - OshiShare integration for file distribution

## Features

### Free Hosting
- No cost for basic server hosting
- Community-supported infrastructure
- Designed for VR creators and their communities

### Easy Management
- Web-based control panel
- One-click server deployment
- Automated backups (best-effort)
- Simple configuration options

### Resonite Integration
- Specialized support for Resonite headless servers
- Repair tools for common issues
- Session management features
- World hosting capabilities

## Getting Started

1. **Sign Up** - Create your account at AMP Panel
2. **Select Server Type** - Choose from available server options
3. **Configure** - Set up your server parameters
4. **Deploy** - Launch your server with one click
5. **Manage** - Use the web panel for ongoing management

## Limitations

- Resources are limited based on availability
- Best-effort service with no guaranteed uptime
- Users responsible for their own backups
- Subject to Terms of Service
